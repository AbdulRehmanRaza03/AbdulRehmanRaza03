## Flagship Project — TheEleven

An AI Operating Assistant designed as a practical, real-world alternative to chatbot-style AI — an operating layer, not a chat window.

| Capability | Description |
|---|---|
| Voice | Real-time voice conversations (speech-in, speech-out) |
| Vision | Screen understanding and visual analysis |
| Memory | Long-term memory for personalized, contextual interaction |
| Research | Intelligent web search and information retrieval |
| Control | Operates applications, files, browser, and terminal directly |
| Autonomy | Executes multi-step tasks without step-by-step supervision |
| Platform | Cross-platform — Windows, macOS, Linux |

**Vision:** an AI companion that manages daily workflows, system tasks, and complex multi-step execution on a computer with minimal human intervention — understanding context, learning continuously, and acting proactively rather than reactively.

*Status: actively in development.*
